# Session Notes

## Talk

Build. Observe. Scale.

Autonomous AI Agents with ADK, MCP Toolbox and Gemini Enterprise Agent Platform.

## Event

AI Weekend: Build with AI 2026 by Google Developer Groups Sharjah, hosted alongside MENA Blockchain Week at Hadron Founders Club, Dubai.

## Core Message

Anyone can build an AI agent. The real challenge is operating an AI agent you can trust in production.

## Covered Topics

- Context engineering for agents.
- Grounding agents with enterprise data.
- Tool execution through MCP Toolbox.
- Agent development with ADK.
- Deployment on Gemini Enterprise Agent Platform.
- Observability across logs, traces and metrics.
- Governance, reliability and production readiness.

## Demo

FinBot is a GCP FinOps agent that can query cloud billing exports, explain cost drivers, detect anomalies, forecast spend and recommend optimization actions.
