# Contributing

Thanks for checking out the session resources.

This repository is primarily a speaker resource repo, but issues and suggestions are welcome.

Good contributions:

- Fix setup instructions.
- Improve sample prompts.
- Add safer demo data patterns.
- Add observability examples.
- Improve MCP Toolbox sample configuration.

Please do not submit real billing data, credentials, project IDs, billing account IDs or customer-specific details.
